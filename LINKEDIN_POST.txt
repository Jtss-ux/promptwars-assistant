Hey everyone! 🚀

I'm incredibly excited to share my submission for the #PromptWars Virtual Hackathon: **LogicFlow - Code Review Assistant** 🤖!

As developers, we've all experienced the frustration of generic AI coding answers or, worse, hallucinated APIs. I wanted to build an AI mentor that actually *understands* development context and validates its facts.

LogicFlow is a context-aware developer assistant with a premium, sleek glassmorphism UI. You select your current task (like "System Architecture" or "Bug Detection"), and the Node.js backend dynamically builds a robust prompt matrix to instruct the AI to behave like a senior architect.

💡 The coolest part? Integrating **Meaningful Google Services**. LogicFlow utilizes Google's **Vertex AI (Gemini 2.5 Flash)** augmented with native **Google Search Grounding**, allowing the model to fact-check its code reviews against the latest internet documentation in real-time, completely preventing hallucinated API syntax. It also features 1-click Markdown exports for team productivity!

All successfully containerized and deployed flawlessly to Google Cloud Run! ☁️
